# Large-UniDet
A practice for million-scale multi-domain universal object detection.   
The 2nd place (**IFFF_RVC**) in ECCV 2022 [Robust Vision Challenge (RVC 2022)](http://www.robustvision.net/leaderboard.php?benchmark=object).
